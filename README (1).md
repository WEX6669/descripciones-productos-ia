# Generador de descripciones de productos con IA

Herramienta web que genera descripciones de productos optimizadas para diferentes plataformas usando IA.

## Funciones

- Adaptado para tienda online, Instagram, Amazon y Wallapop
- Tres tonos: profesional, cercano y premium
- Campo de público objetivo para personalizar el mensaje
- Copia la descripción con un clic

## Tecnología

- HTML, CSS y JavaScript puro (sin frameworks)
- API de Claude (Anthropic)

## Cómo usarlo

1. Clona el repositorio
2. Abre `index.html`
3. Añade tu API key de Anthropic en la variable `API_KEY`
4. ¡Listo!
